# notifications

Notifications is a web application where is displayed a single notification page

![notifications](https://user-images.githubusercontent.com/58858236/204102513-8f1fb068-9792-449f-b92e-0b9557782d38.PNG)

## How to run the project locally
1. Navigate to the project's root directory

2. Install the project's dependencies
```bash
npm install
```

3. Run the application
```bash
npm run dev
```
